# Konzert-Bingo 🎸

Ein interaktives Bingo-Spiel für Konzertbesucher mit typischen Konzert-Phrasen und -Aktionen.

## Einleitung

Konzert-Bingo ist ein unterhaltsames Spiel, das du während eines Konzerts spielen kannst. Markiere die Felder mit Phrasen, die du während des Konzerts hörst oder siehst. Wer zuerst eine vollständige Zeile oder Spalte hat, gewinnt!

## Bedienungsanweisung

1. **Spiel starten**: Öffne einfach die `index.html` Datei in deinem Webbrowser
2. **Spiel vorbereiten**: Das Spiel erstellt automatisch ein 4x4 Bingo-Raster mit zufällig gemischten Konzert-Phrasen
3. **Felder markieren**: Klicke auf ein Feld, wenn du die entsprechende Phrase hörst oder die Aktion siehst
4. **Gewinnen**: Sobald du eine vollständige Zeile oder Spalte markiert hast, werden die Gewinnfelder gelb hervorgehoben

## Enthaltene Phrasen

Das Spiel enthält 69 verschiedene Konzert-typische Phrasen wie:
- "Ihr seid die geilsten!"
- "Lasst uns noch einen Selfie machen"
- "Alle klatschen!"
- "Nazis raus"
- "👏"
- "Hey Hey"
- Und viele mehr...

## Technische Details

- **Technologie**: HTML5, CSS3, Vanilla JavaScript (ES6 Modules)
- **Schriftart**: Teko (Variable Font)
- **Struktur**:
  - `index.html` - Haupteinstiegspunkt
  - `assets/js/` - JavaScript-Module
  - `assets/css/` - Styling
  - `assets/fonts/` - Schriftarten
  - `assets/img/` - Bilder (QR-Code)

## Installation

Keine Installation erforderlich! Einfach die `index.html` Datei in einem modernen Webbrowser öffnen.

## Entwicklung

Falls du das Spiel erweitern oder anpassen möchtest:

1. Klone das Repository oder lade die Dateien herunter
2. Öffne das Projekt in deiner bevorzugten IDE
3. Bearbeite die Dateien:
   - `assets/js/data.js` - Zum Hinzufügen/Ändern von Phrasen
   - `assets/js/settings.js` - Zum Anpassen der Rastergröße
   - `assets/css/styles.css` - Zum Anpassen des Designs
